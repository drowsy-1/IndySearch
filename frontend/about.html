ABOUT INDYSEARCH

IndySearch is a full-text search engine for Neocities — the indie web hosting platform home to hundreds of thousands of personal websites, art projects, zines, and creative experiments.

Unlike conventional search engines, IndySearch is built specifically to surface content from the indie web.

How it works

IndySearch discovers sites by scanning the Neocities browse pages, then looks at each site respecting robots.txt. Page text is extracted, compressed, and stored. A full-text search index is built using Tantivy, with relevance ranking that boosts titles and site names. Search results include highlighted snippets showing exactly where your query matches.

Tech stack

  • Tantivy — Rust-based full-text search engine
  • FastAPI — Python async search API
  • PostgreSQL — site and document metadata
  • Trafilatura — HTML to text extraction
  • Zstandard — compressed text storage
